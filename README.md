# 🎮 Jeu OXO (Tic-Tac-Toe)

## 📌 Description
Ce projet est une implémentation du célèbre jeu **OXO (ou Tic-Tac-Toe)**.

Le jeu propose plusieurs fonctionnalités pour améliorer l'expérience :
- Choix de la taille de la grille
- Mode **joueur vs joueur** (sur le même ordinateur)
- Mode **joueur vs ordinateur** avec une IA qui tente de gagner

---

## ⚙️ Fonctionnalités

- ✅ Grille personnalisable (nombre de cases configurable)
- ✅ Mode 2 joueurs local
- ✅ Mode contre l'ordinateur
- ✅ IA capable de prendre des décisions pour essayer de gagner
- ✅ Interface simple et facile à utiliser

---

## 🕹️ Comment jouer

1. Lancer le programme  
2. Choisir :
   - La taille de la grille (ex : 3x3, 4x4, etc.)
   - Le mode de jeu :
     - Joueur vs Joueur
     - Joueur vs Ordinateur  
3. Jouer chacun son tour en plaçant son symbole (**X** ou **O**)  
4. Le premier joueur à aligner ses symboles gagne  

---

## 🤖 Intelligence Artificielle

En mode joueur vs ordinateur :
- L'ordinateur analyse le jeu
- Il tente de bloquer le joueur
- Il cherche aussi à créer des opportunités de victoire
